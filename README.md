# GatherAway v1.2.5
## Features

Hit the "X" on trees, Always hit the "shinny-spot" on the Ore Nodes (minigames).  

    -Updated to be 24 hours no time restrictions.  
    -Updated the Blacklist  / Whitelist to be easier with examples.
    -Add permission.use and permission.bypass (gatheraway.use & gatherway.bypass)
```sh   
if "Blacklist" = TRUE 
    Tools NOT be able to automatically hit the mini-game "X" or "Shiny-spot"

if "Whitelist" = TRUE
    only the listed tools can auto-hit the mini-game spot.

if "Blacklist" & "Whitelist" = FALSE 
    All-tools will auto-hit the mini-game spot.
```
Debug mode is false by default.  
Deubg mode: TRUE gives verbose messages in the console.
